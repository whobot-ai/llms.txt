## 什么是 llms.txt？

[llms.txt](https://llmstxt.org) 是一项为网站和组织提供 LLM 友好信息的标准提案。它帮助 AI 模型快速理解一家公司做什么、有哪些产品和能力。

## 使用方式

将文件部署到网站根目录：

```
https://www.whobot.com/llms.txt
```

也可以直接引用本仓库中的文件。
